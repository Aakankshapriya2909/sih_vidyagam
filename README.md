# VidyaGam - Interactive Educational Gaming Platform

VidyaGam is a gamified learning platform designed to make education engaging and fun for students through interactive games and quizzes.

## Features

- 🎮 Game-Based Learning
- 📚 Multiple Subjects (Math, Science, GK)
- 🏆 Progress Tracking & Leaderboards
- 👨‍🏫 Expert Instructor Content
- 🌍 Multi-language Support
- 📱 Responsive Design

## Installation

1. Clone the repository:
   git clone https://github.com/Aakankshapriya2909/sih_vidyagam.git

2. Navigate to the project folder:
   cd SIH_VidyaGam

3. Install dependencies:
   npm install

4. Create `.env` file from `.env.example`:
   cp .env.example .env

5. Start the server:
   npm start

   For development with auto-reload:
   npm run dev

6. Open your browser and go to:
   http://localhost:3000

## Technologies Used

- Backend: Node.js, Express.js
- Frontend: HTML5, CSS3, JavaScript (ES6+)
- Icons: Font Awesome
- Architecture: Component-based modular design
