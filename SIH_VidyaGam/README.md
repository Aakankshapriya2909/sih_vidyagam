# VidyaGam - Interactive Educational Gaming Platform

VidyaGam is a gamified learning platform designed to make education engaging and fun for students through interactive games and quizzes.

## Features

- 🎮 Game-Based Learning
- 📚 Multiple Subjects (Math, Science, GK)
- 🏆 Progress Tracking & Leaderboards
- 👨‍🏫 Expert Instructor Content
- 🌍 Multi-language Support
- 📱 Responsive Design

## Installation

1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```

3. Create `.env` file from `.env.example`:
   ```bash
   cp .env.example .env
   ```

4. Start the server:
   ```bash
   npm start
   ```
   
   For development with auto-reload:
   ```bash
   npm run dev
   ```

5. Open your browser and navigate to:
   ```
   http://localhost:3000
   ```

## Technologies Used

- **Backend:** Node.js, Express.js
- **Frontend:** HTML5, CSS3, JavaScript (ES6+)
- **Icons:** Font Awesome
- **Architecture:** Component-based modular design
